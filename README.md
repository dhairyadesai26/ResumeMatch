# 🚀 ResumeMatch AI — ATS-Powered Resume Intelligence Platform

> **Stop guessing why your resume gets rejected. Start optimizing with AI.**

ResumeMatch AI is a modern AI-powered resume analysis platform that evaluates resumes against specific job descriptions, simulates ATS screening, identifies weaknesses, and provides actionable recommendations to improve interview chances.

Built with **React 19, React Router 7, TypeScript, Tailwind CSS, Zustand, PDF Processing, and Puter AI Cloud Services**, ResumeMatch transforms a static PDF resume into an intelligent career optimization experience.

---



## ✨ Key Features

### 🤖 AI Resume Analysis

* Upload PDF resumes instantly
* AI extracts and understands resume content
* Context-aware evaluation against job requirements
* Personalized improvement recommendations

### 🎯 ATS Compatibility Scanner

* Simulates Applicant Tracking Systems (ATS)
* Identifies keyword gaps
* Detects formatting issues
* Evaluates resume readability

### 📊 Intelligent Scoring Engine

* Resume Match Score
* ATS Compliance Score

* Experience Alignment Score

### 📄 PDF Resume Processing

* PDF Upload Support
* PDF-to-Image Conversion

### 🧠 Job Description Matching

* Software Engineer roles

* Any custom job description

### 🔍 Detailed Feedback Reports

Get insights on:

* Professional impact
* ATS ranking potential

### ☁️ Cloud-Native Architecture

Powered by Puter Cloud Services:

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/resumematch-ai.git
cd resumematch-ai
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

Application starts at:

```text
http://localhost:5173
```

---

## Production Build

```bash
npm run build
```

Run production server:

```bash
npm run start
```

---

# 🐳 Docker Support

Build image:

```bash
docker build -t resumematch-ai .
```

Run container:

```bash
docker run -p 3000:3000 resumematch-ai
```
